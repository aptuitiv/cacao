Container
=========

A page container.

### Settings:

- `$container-width`
- `$container-gutter`

### Modifiers

- `-fixed`
- `-gutter-none`
- `-align-left`


Usage
-----

    <div class="header">
        <div class="container -gutter-none">
            [ ... ]
        </div>
    </div>

    <div class="container">
        [ ... ]
    </div>
