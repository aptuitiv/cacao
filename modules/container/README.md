Container
=========

A page container.

### Settings:

- `$container-width`
- `$container-gutter`

### Modifiers

- `--widthFixed`
- `--gutterNone`
- `--alignLeft`


Usage
-----

    <div class="Container">
        [ ... ]
    </div>

    <!-- within another element -->
    <div id="Header">
        <div class="Container Container--gutterNone">
            [ ... ]
        </div>
    </div>
