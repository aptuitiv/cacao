Lists
=====

Generic list styles.

### Settings

- `$list-spaced-margin`
- `$list-barred-margin`
- `$list-dotted-margin`


Usage
-----

### List Reset

    <ul class="list">
        [...]
    </ul>

### Inline List

    <ul class="list-inline">
        [...]
    </ul>

### Spaced Inline List

    <ul class="list-spaced">
        [...]
    </ul>

### Barred Inline List

    <ul class="list-barred">
        [...]
    </ul>

### Dotted Inline List

    <ul class="list-dotted">
        [...]
    </ul>
