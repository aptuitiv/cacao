Lists
=====

Generic list styles.

### Settings

- `$list-spaced-margin`
- `$list-barred-margin`
- `$list-dotted-margin`


Usage
-----

### List Reset

    <ul class="List">
        [...]
    </ul>

### Inline List

    <ul class="List List--inline">
        [...]
    </ul>

### Spaced Inline List

    <ul class="List List--inline List--inline--spaced">
        [...]
    </ul>

### Barred Inline List

    <ul class="List List--inline List--inline--barred">
        [...]
    </ul>

### Dotted Inline List

    <ul class="List List--inline List--inline--dotted">
        [...]
    </ul>
