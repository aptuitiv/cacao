---
title: Padding - bottom
---

These padding styles set the bottom padding on elements.

The size of the padding is set with the [padding variables](/styles/padding/variables).

Each padding size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pb-2-3xs
pb-2-2xs
pb-2-xs
pb-2-sm
pb-2-md
pb-2-lg
pb-2-xl
pb-2-1xl
pb-2-3xl
pb-2-4xl
pb-2-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/padding/bottom.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/padding/3xs/bottom.css';
@import 'cacao-css/dist/padding/2xs/bottom.css';
@import 'cacao-css/dist/padding/xs/bottom.css';
@import 'cacao-css/dist/padding/sm/bottom.css';
@import 'cacao-css/dist/padding/md/bottom.css';
@import 'cacao-css/dist/padding/lg/bottom.css';
@import 'cacao-css/dist/padding/xl/bottom.css';
@import 'cacao-css/dist/padding/2xl/bottom.css';
@import 'cacao-css/dist/padding/3xl/bottom.css';
@import 'cacao-css/dist/padding/4xl/bottom.css';
@import 'cacao-css/dist/padding/5xl/bottom.css';
```
