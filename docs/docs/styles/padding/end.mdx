---
title: Padding - end (right)
---

These padding styles set the end padding. In left-to-right languages this will set the right padding.

The size of the padding is set with the [padding variables](/styles/padding/variables).

Each padding size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pe-2-3xs
pe-2-2xs
pe-2-xs
pe-2-sm
pe-2-md
pe-2-lg
pe-2-xl
pe-2-1xl
pe-2-3xl
pe-2-4xl
pe-2-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/padding/end.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/padding/3xs/end.css';
@import 'cacao-css/dist/padding/2xs/end.css';
@import 'cacao-css/dist/padding/xs/end.css';
@import 'cacao-css/dist/padding/sm/end.css';
@import 'cacao-css/dist/padding/md/end.css';
@import 'cacao-css/dist/padding/lg/end.css';
@import 'cacao-css/dist/padding/xl/end.css';
@import 'cacao-css/dist/padding/2xl/end.css';
@import 'cacao-css/dist/padding/3xl/end.css';
@import 'cacao-css/dist/padding/4xl/end.css';
@import 'cacao-css/dist/padding/5xl/end.css';
```
