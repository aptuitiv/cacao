---
title: Padding - horizontal and vertical
---

These padding styles set the paddings on the top, right, bottom, and left sides of an element.

The size of the padding is set with the [padding variables](/styles/padding/variables).

Each padding size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
p-2-3xs
p-2-2xs
p-2-xs
p-2-sm
p-2-md
p-2-lg
p-2-xl
p-2-1xl
p-2-3xl
p-2-4xl
p-2-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/padding/all.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/padding/3xs/all.css';
@import 'cacao-css/dist/padding/2xs/all.css';
@import 'cacao-css/dist/padding/xs/all.css';
@import 'cacao-css/dist/padding/sm/all.css';
@import 'cacao-css/dist/padding/md/all.css';
@import 'cacao-css/dist/padding/lg/all.css';
@import 'cacao-css/dist/padding/xl/all.css';
@import 'cacao-css/dist/padding/2xl/all.css';
@import 'cacao-css/dist/padding/3xl/all.css';
@import 'cacao-css/dist/padding/4xl/all.css';
@import 'cacao-css/dist/padding/5xl/all.css';
```
