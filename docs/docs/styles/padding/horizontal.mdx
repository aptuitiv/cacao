---
title: Padding - horizontal
---

These padding styles set the left and right paddings.

The size of the padding is set with the [padding variables](/styles/padding/variables).

Each padding size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
px-2-3xs
px-2-2xs
px-2-xs
px-2-sm
px-2-md
px-2-lg
px-2-xl
px-2-1xl
px-2-3xl
px-2-4xl
px-2-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/padding/horizontal.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/padding/3xs/horizontal.css';
@import 'cacao-css/dist/padding/2xs/horizontal.css';
@import 'cacao-css/dist/padding/xs/horizontal.css';
@import 'cacao-css/dist/padding/sm/horizontal.css';
@import 'cacao-css/dist/padding/md/horizontal.css';
@import 'cacao-css/dist/padding/lg/horizontal.css';
@import 'cacao-css/dist/padding/xl/horizontal.css';
@import 'cacao-css/dist/padding/2xl/horizontal.css';
@import 'cacao-css/dist/padding/3xl/horizontal.css';
@import 'cacao-css/dist/padding/4xl/horizontal.css';
@import 'cacao-css/dist/padding/5xl/horizontal.css';
```
