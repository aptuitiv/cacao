---
title: Padding
---

The padding styles set paddinggs around elements.

There are seven types of padding styles.

1. Both [horizontal and vertical paddings](/styles/padding/all)
2. [Horizontal paddings](/styles/padding/horizontal)
3. [Vertical paddings](/styles/padding/vertical)
4. [Padding at the top](/styles/padding/top)
5. [Padding at the end](/styles/padding/end) (this would be the right when text is displayed left-to-right)
6. [Padding at the bottom](/styles/padding/bottom)
7. [Padding at the start](/styles/padding/start) (this would be the left when text is displayed left-to-right)

The size of the padding is set with the [padding variables](/styles/padding/variables).

## Combined imports

If you will use all of the padding styles then you can use the combined import file.

```css
@import 'cacao-css/dist/padding/combined-import.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/padding/3xs/combined-import.css';
@import 'cacao-css/dist/padding/2xs/combined-import.css';
@import 'cacao-css/dist/padding/xs/combined-import.css';
@import 'cacao-css/dist/padding/sm/combined-import.css';
@import 'cacao-css/dist/padding/md/combined-import.css';
@import 'cacao-css/dist/padding/lg/combined-import.css';
@import 'cacao-css/dist/padding/xl/combined-import.css';
@import 'cacao-css/dist/padding/2xl/combined-import.css';
@import 'cacao-css/dist/padding/3xl/combined-import.css';
@import 'cacao-css/dist/padding/4xl/combined-import.css';
@import 'cacao-css/dist/padding/5xl/combined-import.css';
```
