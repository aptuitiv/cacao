---
title: Padding - top
---

These padding styles set the top padding on elements.

The size of the padding is set with the [padding variables](/styles/padding/variables).

Each padding size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pt-2-3xs
pt-2-2xs
pt-2-xs
pt-2-sm
pt-2-md
pt-2-lg
pt-2-xl
pt-2-1xl
pt-2-3xl
pt-2-4xl
pt-2-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/padding/top.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/padding/3xs/top.css';
@import 'cacao-css/dist/padding/2xs/top.css';
@import 'cacao-css/dist/padding/xs/top.css';
@import 'cacao-css/dist/padding/sm/top.css';
@import 'cacao-css/dist/padding/md/top.css';
@import 'cacao-css/dist/padding/lg/top.css';
@import 'cacao-css/dist/padding/xl/top.css';
@import 'cacao-css/dist/padding/2xl/top.css';
@import 'cacao-css/dist/padding/3xl/top.css';
@import 'cacao-css/dist/padding/4xl/top.css';
@import 'cacao-css/dist/padding/5xl/top.css';
```

See [Import Cacao CSS](/import) for more information.
