---
title: Display order
---

The display order styles provide helper classes for setting the display order of items with the `order` style.

Each display order base class has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
order-2-3xs
order-2-2xs
order-2-xs
order-2-sm
order-2-md
order-2-lg
order-2-xl
order-2-1xl
order-2-3xl
order-2-4xl
order-2-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/display/order.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/display/3xs/order.css';
@import 'cacao-css/dist/display/2xs/order.css';
@import 'cacao-css/dist/display/xs/order.css';
@import 'cacao-css/dist/display/sm/order.css';
@import 'cacao-css/dist/display/md/order.css';
@import 'cacao-css/dist/display/lg/order.css';
@import 'cacao-css/dist/display/xl/order.css';
@import 'cacao-css/dist/display/2xl/order.css';
@import 'cacao-css/dist/display/3xl/order.css';
@import 'cacao-css/dist/display/4xl/order.css';
@import 'cacao-css/dist/display/5xl/order.css';
```

See [Import Cacao CSS](/import) for more information.
