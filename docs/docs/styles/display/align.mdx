---
title: Aligning flexbox items
---

The align styles handle aligning flexbox items with `align-items`.

Each align base class has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
align-start-3xs
align-start-2xs
align-start-xs
align-start-sm
align-start-md
align-start-lg
align-start-xl
align-start-1xl
align-start-3xl
align-start-4xl
align-start-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/display/align.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/display/3xs/align.css';
@import 'cacao-css/dist/display/2xs/align.css';
@import 'cacao-css/dist/display/xs/align.css';
@import 'cacao-css/dist/display/sm/align.css';
@import 'cacao-css/dist/display/md/align.css';
@import 'cacao-css/dist/display/lg/align.css';
@import 'cacao-css/dist/display/xl/align.css';
@import 'cacao-css/dist/display/2xl/align.css';
@import 'cacao-css/dist/display/3xl/align.css';
@import 'cacao-css/dist/display/4xl/align.css';
@import 'cacao-css/dist/display/5xl/align.css';
```

See [Import Cacao CSS](/import) for more information.
