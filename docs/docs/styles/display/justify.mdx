---
title: Justify flexbox items
---

The justify utility styles currently handle `justify-content` styles.

Each justify base class has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
justify-start-3xs
justify-start-2xs
justify-start-xs
justify-start-sm
justify-start-md
justify-start-lg
justify-start-xl
justify-start-1xl
justify-start-3xl
justify-start-4xl
justify-start-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/display/justify.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/display/3xs/justify.css';
@import 'cacao-css/dist/display/2xs/justify.css';
@import 'cacao-css/dist/display/xs/justify.css';
@import 'cacao-css/dist/display/sm/justify.css';
@import 'cacao-css/dist/display/md/justify.css';
@import 'cacao-css/dist/display/lg/justify.css';
@import 'cacao-css/dist/display/xl/justify.css';
@import 'cacao-css/dist/display/2xl/justify.css';
@import 'cacao-css/dist/display/3xl/justify.css';
@import 'cacao-css/dist/display/4xl/justify.css';
@import 'cacao-css/dist/display/5xl/justify.css';
```
