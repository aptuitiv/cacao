---
title: Push - eights
---

This provides styles for pushs using eighths.

The following sizes are available:

- push-1-8
- push-2-8
- push-3-8
- push-4-8
- push-5-8
- push-6-8
- push-7-8
- push-8-8

Each push size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
push-1-8-3xs
push-1-8-2xs
push-1-8-xs
push-1-8-sm
push-1-8-md
push-1-8-lg
push-1-8-xl
push-1-8-2xl
push-1-8-3xl
push-1-8-4xl
push-1-8-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/push/push-8.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/push/3xs/push-8.css';
@import 'cacao-css/dist/push/2xs/push-8.css';
@import 'cacao-css/dist/push/xs/push-8.css';
@import 'cacao-css/dist/push/sm/push-8.css';
@import 'cacao-css/dist/push/md/push-8.css';
@import 'cacao-css/dist/push/lg/push-8.css';
@import 'cacao-css/dist/push/xl/push-8.css';
@import 'cacao-css/dist/push/2xl/push-8.css';
@import 'cacao-css/dist/push/3xl/push-8.css';
@import 'cacao-css/dist/push/4xl/push-8.css';
@import 'cacao-css/dist/push/5xl/push-8.css';
```

See [Import Cacao CSS](/import) for more information.
