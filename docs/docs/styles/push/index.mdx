title: Push
---

The push classes move an element to the right by setting the `left` style. The elements are also giving a `position` of `relative`.

The following push sizes are available:

- [No push](/styles/push/zero)
- [Full push](/styles/push/one)
- [Halfs](/styles/push/two)
- [Thirds](/styles/push/three)
- [Fourths](/styles/push/four)
- [Fifths](/styles/push/five)
- [Sixths](/styles/push/six)
- [Sevenths](/styles/push/seven)
- [Eighths](/styles/push/eight)
- [Nineths](/styles/push/nine)
- [Tenths](/styles/push/ten)

## Combined imports

If you will use all of the push sizes then you can use the combined import file.

```css
@import 'cacao-css/dist/push/combined-import.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/push/3xs/combined-import.css';
@import 'cacao-css/dist/push/2xs/combined-import.css';
@import 'cacao-css/dist/push/xs/combined-import.css';
@import 'cacao-css/dist/push/sm/combined-import.css';
@import 'cacao-css/dist/push/md/combined-import.css';
@import 'cacao-css/dist/push/lg/combined-import.css';
@import 'cacao-css/dist/push/xl/combined-import.css';
@import 'cacao-css/dist/push/2xl/combined-import.css';
@import 'cacao-css/dist/push/3xl/combined-import.css';
@import 'cacao-css/dist/push/4xl/combined-import.css';
@import 'cacao-css/dist/push/5xl/combined-import.css';
```
