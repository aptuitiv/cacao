---
title: Full push
---

This provides styles for setting an element to be full push (i.e. `left: 100%`).

The following sizeis available:

- push-1-1

Each push size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
push-1-1-3xs
push-1-1-2xs
push-1-1-xs
push-1-1-sm
push-1-1-md
push-1-1-lg
push-1-1-xl
push-1-1-1xl
push-1-1-3xl
push-1-1-4xl
push-1-1-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/push/push-1.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/push/3xs/push-1.css';
@import 'cacao-css/dist/push/2xs/push-1.css';
@import 'cacao-css/dist/push/xs/push-1.css';
@import 'cacao-css/dist/push/sm/push-1.css';
@import 'cacao-css/dist/push/md/push-1.css';
@import 'cacao-css/dist/push/lg/push-1.css';
@import 'cacao-css/dist/push/xl/push-1.css';
@import 'cacao-css/dist/push/2xl/push-1.css';
@import 'cacao-css/dist/push/3xl/push-1.css';
@import 'cacao-css/dist/push/4xl/push-1.css';

```
