---
title: Push - fourths
---

This provides styles for pushs using fourths.

The following sizes are available:

- push-1-4
- push-2-4
- push-3-4
- push-4-4

Each push size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
push-1-4-3xs
push-1-4-2xs
push-1-4-xs
push-1-4-sm
push-1-4-md
push-1-4-lg
push-1-4-xl
push-1-4-2xl
push-1-4-3xl
push-1-4-4xl
push-1-4-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/push/push-4.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/push/3xs/push-4.css';
@import 'cacao-css/dist/push/2xs/push-4.css';
@import 'cacao-css/dist/push/xs/push-4.css';
@import 'cacao-css/dist/push/sm/push-4.css';
@import 'cacao-css/dist/push/md/push-4.css';
@import 'cacao-css/dist/push/lg/push-4.css';
@import 'cacao-css/dist/push/xl/push-4.css';
@import 'cacao-css/dist/push/2xl/push-4.css';
@import 'cacao-css/dist/push/3xl/push-4.css';
@import 'cacao-css/dist/push/4xl/push-4.css';
@import 'cacao-css/dist/push/5xl/push-4.css';
```
