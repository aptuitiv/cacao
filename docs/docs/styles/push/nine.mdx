---
title: Push - nineths
---

This provides styles for pushs using ninths.

The following sizes are available:

- push-1-9
- push-2-9
- push-3-9
- push-4-9
- push-5-9
- push-6-9
- push-7-9
- push-8-9
- push-9-9

Each push size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
push-1-9-3xs
push-1-9-2xs
push-1-9-xs
push-1-9-sm
push-1-9-md
push-1-9-lg
push-1-9-xl
push-1-9-2xl
push-1-9-3xl
push-1-9-4xl
push-1-9-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/push/push-9.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/push/3xs/push-9.css';
@import 'cacao-css/dist/push/2xs/push-9.css';
@import 'cacao-css/dist/push/xs/push-9.css';
@import 'cacao-css/dist/push/sm/push-9.css';
@import 'cacao-css/dist/push/md/push-9.css';
@import 'cacao-css/dist/push/lg/push-9.css';
@import 'cacao-css/dist/push/xl/push-9.css';
@import 'cacao-css/dist/push/2xl/push-9.css';
@import 'cacao-css/dist/push/3xl/push-9.css';
@import 'cacao-css/dist/push/4xl/push-9.css';
@import 'cacao-css/dist/push/5xl/push-9.css';
```
