---
title: Push - fifths
---

This provides styles for pushs using fifths.

The following sizes are available:

- push-1-5
- push-2-5
- push-3-5
- push-4-5
- push-5-5

Each push size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
push-1-5-3xs
push-1-5-2xs
push-1-5-xs
push-1-5-sm
push-1-5-md
push-1-5-lg
push-1-5-xl
push-1-5-2xl
push-1-5-3xl
push-1-5-4xl
push-1-5-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/push/push-5.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/push/3xs/push-5.css';
@import 'cacao-css/dist/push/2xs/push-5.css';
@import 'cacao-css/dist/push/xs/push-5.css';
@import 'cacao-css/dist/push/sm/push-5.css';
@import 'cacao-css/dist/push/md/push-5.css';
@import 'cacao-css/dist/push/lg/push-5.css';
@import 'cacao-css/dist/push/xl/push-5.css';
@import 'cacao-css/dist/push/2xl/push-5.css';
@import 'cacao-css/dist/push/3xl/push-5.css';
@import 'cacao-css/dist/push/4xl/push-5.css';
@import 'cacao-css/dist/push/5xl/push-5.css';
```
