---
title: No push
---

This provides styles for setting element to have no push (i.e. `left: 0`).

The following class is available:

- push-0

The push base class has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
push-0-3xs
push-0-2xs
push-0-xs
push-0-sm
push-0-md
push-0-lg
push-0-xl
push-0-1xl
push-0-3xl
push-0-4xl
push-0-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/push/push-0.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/push/3xs/push-0.css';
@import 'cacao-css/dist/push/2xs/push-0.css';
@import 'cacao-css/dist/push/xs/push-0.css';
@import 'cacao-css/dist/push/sm/push-0.css';
@import 'cacao-css/dist/push/md/push-0.css';
@import 'cacao-css/dist/push/lg/push-0.css';
@import 'cacao-css/dist/push/xl/push-0.css';
@import 'cacao-css/dist/push/2xl/push-0.css';
@import 'cacao-css/dist/push/3xl/push-0.css';
@import 'cacao-css/dist/push/4xl/push-0.css';

```
