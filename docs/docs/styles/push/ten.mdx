---
title: Push - tenths
---

This provides styles for pushs using tenths.

The following sizes are available:

- push-1-10
- push-2-10
- push-3-10
- push-4-10
- push-5-10
- push-6-10
- push-7-10
- push-8-10
- push-9-10
- push-10-10

Each push size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
push-1-10-3xs
push-1-10-2xs
push-1-10-xs
push-1-10-sm
push-1-10-md
push-1-10-lg
push-1-10-xl
push-1-10-2xl
push-1-10-3xl
push-1-10-4xl
push-1-10-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/push/push-10.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/push/3xs/push-10.css';
@import 'cacao-css/dist/push/2xs/push-10.css';
@import 'cacao-css/dist/push/xs/push-10.css';
@import 'cacao-css/dist/push/sm/push-10.css';
@import 'cacao-css/dist/push/md/push-10.css';
@import 'cacao-css/dist/push/lg/push-10.css';
@import 'cacao-css/dist/push/xl/push-10.css';
@import 'cacao-css/dist/push/2xl/push-10.css';
@import 'cacao-css/dist/push/3xl/push-10.css';
@import 'cacao-css/dist/push/4xl/push-10.css';
@import 'cacao-css/dist/push/5xl/push-10.css';
```

See [Import Cacao CSS](/import) for more information.
