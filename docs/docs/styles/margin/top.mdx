---
title: Margins - top
---

These margin styles set the top margin on elements.

The size of the margin is set with the [margin variables](/styles/margin/variables).

Each margin size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
mt-2-3xs
mt-2-2xs
mt-2-xs
mt-2-sm
mt-2-md
mt-2-lg
mt-2-xl
mt-2-1xl
mt-2-3xl
mt-2-4xl
mt-2-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/margin/top.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/margin/3xs/top.css';
@import 'cacao-css/dist/margin/2xs/top.css';
@import 'cacao-css/dist/margin/xs/top.css';
@import 'cacao-css/dist/margin/sm/top.css';
@import 'cacao-css/dist/margin/md/top.css';
@import 'cacao-css/dist/margin/lg/top.css';
@import 'cacao-css/dist/margin/xl/top.css';
@import 'cacao-css/dist/margin/2xl/top.css';
@import 'cacao-css/dist/margin/3xl/top.css';
@import 'cacao-css/dist/margin/4xl/top.css';
@import 'cacao-css/dist/margin/5xl/top.css';
```

See [Import Cacao CSS](/import) for more information.
