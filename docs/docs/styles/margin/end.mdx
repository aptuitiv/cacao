---
title: Margins - end (right)
---

These margin styles set the end margin. In left-to-right languages this will set the right margin.

The size of the margin is set with the [margin variables](/styles/margin/variables).

Each margin size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
me-2-3xs
me-2-2xs
me-2-xs
me-2-sm
me-2-md
me-2-lg
me-2-xl
me-2-1xl
me-2-3xl
me-2-4xl
me-2-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/margin/end.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/margin/3xs/end.css';
@import 'cacao-css/dist/margin/2xs/end.css';
@import 'cacao-css/dist/margin/xs/end.css';
@import 'cacao-css/dist/margin/sm/end.css';
@import 'cacao-css/dist/margin/md/end.css';
@import 'cacao-css/dist/margin/lg/end.css';
@import 'cacao-css/dist/margin/xl/end.css';
@import 'cacao-css/dist/margin/2xl/end.css';
@import 'cacao-css/dist/margin/3xl/end.css';
@import 'cacao-css/dist/margin/4xl/end.css';
@import 'cacao-css/dist/margin/5xl/end.css';
```

See [Import Cacao CSS](/import) for more information.
