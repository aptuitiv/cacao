---
title: Margins - start (left)
---

These margin styles set the start margin. In left-to-right languages this will set the left margin.

The size of the margin is set with the [margin variables](/styles/margin/variables).

Each margin size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
ms-2-3xs
ms-2-2xs
ms-2-xs
ms-2-sm
ms-2-md
ms-2-lg
ms-2-xl
ms-2-1xl
ms-2-3xl
ms-2-4xl
ms-2-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/margin/start.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/margin/3xs/start.css';
@import 'cacao-css/dist/margin/2xs/start.css';
@import 'cacao-css/dist/margin/xs/start.css';
@import 'cacao-css/dist/margin/sm/start.css';
@import 'cacao-css/dist/margin/md/start.css';
@import 'cacao-css/dist/margin/lg/start.css';
@import 'cacao-css/dist/margin/xl/start.css';
@import 'cacao-css/dist/margin/2xl/start.css';
@import 'cacao-css/dist/margin/3xl/start.css';
@import 'cacao-css/dist/margin/4xl/start.css';
@import 'cacao-css/dist/margin/5xl/start.css';
```

See [Import Cacao CSS](/import) for more information.
