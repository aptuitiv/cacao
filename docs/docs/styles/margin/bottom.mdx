---
title: Margins - bottom
---

These margin styles set the bottom margin on elements.

The size of the margin is set with the [margin variables](/styles/margin/variables).

Each margin size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
mb-2-3xs
mb-2-2xs
mb-2-xs
mb-2-sm
mb-2-md
mb-2-lg
mb-2-xl
mb-2-1xl
mb-2-3xl
mb-2-4xl
mb-2-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/margin/bottom.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/margin/3xs/bottom.css';
@import 'cacao-css/dist/margin/2xs/bottom.css';
@import 'cacao-css/dist/margin/xs/bottom.css';
@import 'cacao-css/dist/margin/sm/bottom.css';
@import 'cacao-css/dist/margin/md/bottom.css';
@import 'cacao-css/dist/margin/lg/bottom.css';
@import 'cacao-css/dist/margin/xl/bottom.css';
@import 'cacao-css/dist/margin/2xl/bottom.css';
@import 'cacao-css/dist/margin/3xl/bottom.css';
@import 'cacao-css/dist/margin/4xl/bottom.css';
@import 'cacao-css/dist/margin/5xl/bottom.css';
```
