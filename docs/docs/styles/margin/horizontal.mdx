---
title: Margins - horizontal
---

These margin styles set the left and right margins.

The size of the margin is set with the [margin variables](/styles/margin/variables).

Each margin size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
mx-2-3xs
mx-2-2xs
mx-2-xs
mx-2-sm
mx-2-md
mx-2-lg
mx-2-xl
mx-2-1xl
mx-2-3xl
mx-2-4xl
mx-2-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/margin/horizontal.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/margin/3xs/horizontal.css';
@import 'cacao-css/dist/margin/2xs/horizontal.css';
@import 'cacao-css/dist/margin/xs/horizontal.css';
@import 'cacao-css/dist/margin/sm/horizontal.css';
@import 'cacao-css/dist/margin/md/horizontal.css';
@import 'cacao-css/dist/margin/lg/horizontal.css';
@import 'cacao-css/dist/margin/xl/horizontal.css';
@import 'cacao-css/dist/margin/2xl/horizontal.css';
@import 'cacao-css/dist/margin/3xl/horizontal.css';
@import 'cacao-css/dist/margin/4xl/horizontal.css';
@import 'cacao-css/dist/margin/5xl/horizontal.css';
```

See [Import Cacao CSS](/import) for more information.
