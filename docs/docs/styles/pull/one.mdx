---
title: Full pull
---

This provides styles for setting an element to be full pull (i.e. `right: 100%`).

The following sizeis available:

- pull-1-1

Each pull size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pull-1-1-3xs
pull-1-1-2xs
pull-1-1-xs
pull-1-1-sm
pull-1-1-md
pull-1-1-lg
pull-1-1-xl
pull-1-1-1xl
pull-1-1-3xl
pull-1-1-4xl
pull-1-1-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/pull/pull-1.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/pull/3xs/pull-1.css';
@import 'cacao-css/dist/pull/2xs/pull-1.css';
@import 'cacao-css/dist/pull/xs/pull-1.css';
@import 'cacao-css/dist/pull/sm/pull-1.css';
@import 'cacao-css/dist/pull/md/pull-1.css';
@import 'cacao-css/dist/pull/lg/pull-1.css';
@import 'cacao-css/dist/pull/xl/pull-1.css';
@import 'cacao-css/dist/pull/2xl/pull-1.css';
@import 'cacao-css/dist/pull/3xl/pull-1.css';
@import 'cacao-css/dist/pull/4xl/pull-1.css';
@import 'cacao-css/dist/pull/5xl/pull-1.css';
```

See [Import Cacao CSS](/import) for more information.
