---
title: Pull - nineths
---

This provides styles for pulls using ninths.

The following sizes are available:

- pull-1-9
- pull-2-9
- pull-3-9
- pull-4-9
- pull-5-9
- pull-6-9
- pull-7-9
- pull-8-9
- pull-9-9

Each pull size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pull-1-9-3xs
pull-1-9-2xs
pull-1-9-xs
pull-1-9-sm
pull-1-9-md
pull-1-9-lg
pull-1-9-xl
pull-1-9-2xl
pull-1-9-3xl
pull-1-9-4xl
pull-1-9-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/pull/pull-9.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/pull/3xs/pull-9.css';
@import 'cacao-css/dist/pull/2xs/pull-9.css';
@import 'cacao-css/dist/pull/xs/pull-9.css';
@import 'cacao-css/dist/pull/sm/pull-9.css';
@import 'cacao-css/dist/pull/md/pull-9.css';
@import 'cacao-css/dist/pull/lg/pull-9.css';
@import 'cacao-css/dist/pull/xl/pull-9.css';
@import 'cacao-css/dist/pull/2xl/pull-9.css';
@import 'cacao-css/dist/pull/3xl/pull-9.css';
@import 'cacao-css/dist/pull/4xl/pull-9.css';
@import 'cacao-css/dist/pull/5xl/pull-9.css';
```

See [Import Cacao CSS](/import) for more information.
