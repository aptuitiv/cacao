---
title: Pull - eights
---

This provides styles for pulls using eighths.

The following sizes are available:

- pull-1-8
- pull-2-8
- pull-3-8
- pull-4-8
- pull-5-8
- pull-6-8
- pull-7-8
- pull-8-8

Each pull size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pull-1-8-3xs
pull-1-8-2xs
pull-1-8-xs
pull-1-8-sm
pull-1-8-md
pull-1-8-lg
pull-1-8-xl
pull-1-8-2xl
pull-1-8-3xl
pull-1-8-4xl
pull-1-8-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/pull/pull-8.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/pull/3xs/pull-8.css';
@import 'cacao-css/dist/pull/2xs/pull-8.css';
@import 'cacao-css/dist/pull/xs/pull-8.css';
@import 'cacao-css/dist/pull/sm/pull-8.css';
@import 'cacao-css/dist/pull/md/pull-8.css';
@import 'cacao-css/dist/pull/lg/pull-8.css';
@import 'cacao-css/dist/pull/xl/pull-8.css';
@import 'cacao-css/dist/pull/2xl/pull-8.css';
@import 'cacao-css/dist/pull/3xl/pull-8.css';
@import 'cacao-css/dist/pull/4xl/pull-8.css';
@import 'cacao-css/dist/pull/5xl/pull-8.css';
```
