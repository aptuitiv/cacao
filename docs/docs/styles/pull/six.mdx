---
title: Pull - sixths
---

This provides styles for pulls using sixths.

The following sizes are available:

- pull-1-6
- pull-2-6
- pull-3-6
- pull-4-6
- pull-5-6
- pull-6-6

Each pull size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pull-1-6-3xs
pull-1-6-2xs
pull-1-6-xs
pull-1-6-sm
pull-1-6-md
pull-1-6-lg
pull-1-6-xl
pull-1-6-2xl
pull-1-6-3xl
pull-1-6-4xl
pull-1-6-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/pull/pull-6.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/pull/3xs/pull-6.css';
@import 'cacao-css/dist/pull/2xs/pull-6.css';
@import 'cacao-css/dist/pull/xs/pull-6.css';
@import 'cacao-css/dist/pull/sm/pull-6.css';
@import 'cacao-css/dist/pull/md/pull-6.css';
@import 'cacao-css/dist/pull/lg/pull-6.css';
@import 'cacao-css/dist/pull/xl/pull-6.css';
@import 'cacao-css/dist/pull/2xl/pull-6.css';
@import 'cacao-css/dist/pull/3xl/pull-6.css';
@import 'cacao-css/dist/pull/4xl/pull-6.css';
@import 'cacao-css/dist/pull/5xl/pull-6.css';
```
