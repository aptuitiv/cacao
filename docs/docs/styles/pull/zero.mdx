---
title: No pull
---

This provides styles for setting element to have no pull (i.e. `right: 0`).

The following class is available:

- pull-0

The pull base class has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pull-0-3xs
pull-0-2xs
pull-0-xs
pull-0-sm
pull-0-md
pull-0-lg
pull-0-xl
pull-0-1xl
pull-0-3xl
pull-0-4xl
pull-0-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/pull/pull-0.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/pull/3xs/pull-0.css';
@import 'cacao-css/dist/pull/2xs/pull-0.css';
@import 'cacao-css/dist/pull/xs/pull-0.css';
@import 'cacao-css/dist/pull/sm/pull-0.css';
@import 'cacao-css/dist/pull/md/pull-0.css';
@import 'cacao-css/dist/pull/lg/pull-0.css';
@import 'cacao-css/dist/pull/xl/pull-0.css';
@import 'cacao-css/dist/pull/2xl/pull-0.css';
@import 'cacao-css/dist/pull/3xl/pull-0.css';
@import 'cacao-css/dist/pull/4xl/pull-0.css';
@import 'cacao-css/dist/pull/5xl/pull-0.css';
```

See [Import Cacao CSS](/import) for more information.
