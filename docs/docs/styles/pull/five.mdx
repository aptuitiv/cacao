---
title: Pull - fifths
---

This provides styles for pulls using fifths.

The following sizes are available:

- pull-1-5
- pull-2-5
- pull-3-5
- pull-4-5
- pull-5-5

Each pull size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
pull-1-5-3xs
pull-1-5-2xs
pull-1-5-xs
pull-1-5-sm
pull-1-5-md
pull-1-5-lg
pull-1-5-xl
pull-1-5-2xl
pull-1-5-3xl
pull-1-5-4xl
pull-1-5-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/pull/pull-5.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/pull/3xs/pull-5.css';
@import 'cacao-css/dist/pull/2xs/pull-5.css';
@import 'cacao-css/dist/pull/xs/pull-5.css';
@import 'cacao-css/dist/pull/sm/pull-5.css';
@import 'cacao-css/dist/pull/md/pull-5.css';
@import 'cacao-css/dist/pull/lg/pull-5.css';
@import 'cacao-css/dist/pull/xl/pull-5.css';
@import 'cacao-css/dist/pull/2xl/pull-5.css';
@import 'cacao-css/dist/pull/3xl/pull-5.css';
@import 'cacao-css/dist/pull/4xl/pull-5.css';
@import 'cacao-css/dist/pull/5xl/pull-5.css';
```

See [Import Cacao CSS](/import) for more information.
