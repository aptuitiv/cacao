title: Pull

The pull classes move an element to the left by setting the `right` style. The elements are also giving a `position` of `relative`.

The following pull sizes are available:

- [No pull](/styles/pull/zero)
- [Full pull](/styles/pull/one)
- [Halfs](/styles/pull/two)
- [Thirds](/styles/pull/three)
- [Fourths](/styles/pull/four)
- [Fifths](/styles/pull/five)
- [Sixths](/styles/pull/six)
- [Sevenths](/styles/pull/seven)
- [Eighths](/styles/pull/eight)
- [Nineths](/styles/pull/nine)
- [Tenths](/styles/pull/ten)

## Combined imports

If you will use all of the pull sizes then you can use the combined import file.

```css
@import 'cacao-css/dist/pull/combined-import.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/pull/3xs/combined-import.css';
@import 'cacao-css/dist/pull/2xs/combined-import.css';
@import 'cacao-css/dist/pull/xs/combined-import.css';
@import 'cacao-css/dist/pull/sm/combined-import.css';
@import 'cacao-css/dist/pull/md/combined-import.css';
@import 'cacao-css/dist/pull/lg/combined-import.css';
@import 'cacao-css/dist/pull/xl/combined-import.css';
@import 'cacao-css/dist/pull/2xl/combined-import.css';
@import 'cacao-css/dist/pull/3xl/combined-import.css';
@import 'cacao-css/dist/pull/4xl/combined-import.css';
@import 'cacao-css/dist/pull/5xl/combined-import.css';
```
