---
title: Grid columns - one column
---

This provides styles for a one-column grid.

This size is often used to indicate that a column will be full width at the smallest screen size.

Below is a basic example:

```html
<div class="grid">
    <div class="col-1-1 col-1-2-md">Column 1</div>
    <div class="col-1-1 col-1-2-md">Column 2</div>
</div>
```

You could, however, have just a one-column layout.

```html
<div class="grid">
    <div class="col-1-1">Column 1</div>
</div>
```

The following sizes are available:

- col-1-1

Each column size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
col-1-1-3xs
col-1-1-2xs
col-1-1-xs
col-1-1-sm
col-1-1-md
col-1-1-lg
col-1-1-xl
col-1-1-1xl
col-1-1-3xl
col-1-1-4xl
col-1-1-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-1.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-1.css';
@import 'cacao-css/dist/grid-column/2xs/col-1.css';
@import 'cacao-css/dist/grid-column/xs/col-1.css';
@import 'cacao-css/dist/grid-column/sm/col-1.css';
@import 'cacao-css/dist/grid-column/md/col-1.css';
@import 'cacao-css/dist/grid-column/lg/col-1.css';
@import 'cacao-css/dist/grid-column/xl/col-1.css';
@import 'cacao-css/dist/grid-column/2xl/col-1.css';
@import 'cacao-css/dist/grid-column/3xl/col-1.css';
@import 'cacao-css/dist/grid-column/4xl/col-1.css';
@import 'cacao-css/dist/grid-column/5xl/col-1.css';
```

See [Import Cacao CSS](/import) for more information.
