---
title: Grid columns - nine column
---

This provides styles for a nine-column grid.

Below is a basic example:

```html
<div class="grid">
    <div class="col-1-3">Column 1</div>
    <div class="col-4-9">Column 2</div>
    <div class="col-2-9">Column 3</div>
</div>
```

The following sizes are available:

- col-1-9
- col-2-9
- col-3-9
- col-4-9
- col-5-9
- col-6-9
- col-7-9
- col-8-9
- col-9-9

Each column size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
col-1-9-3xs
col-1-9-2xs
col-1-9-xs
col-1-9-sm
col-1-9-md
col-1-9-lg
col-1-9-xl
col-1-9-2xl
col-1-9-3xl
col-1-9-4xl
col-1-9-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-9.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-9.css';
@import 'cacao-css/dist/grid-column/2xs/col-9.css';
@import 'cacao-css/dist/grid-column/xs/col-9.css';
@import 'cacao-css/dist/grid-column/sm/col-9.css';
@import 'cacao-css/dist/grid-column/md/col-9.css';
@import 'cacao-css/dist/grid-column/lg/col-9.css';
@import 'cacao-css/dist/grid-column/xl/col-9.css';
@import 'cacao-css/dist/grid-column/2xl/col-9.css';
@import 'cacao-css/dist/grid-column/3xl/col-9.css';
@import 'cacao-css/dist/grid-column/4xl/col-9.css';
@import 'cacao-css/dist/grid-column/5xl/col-9.css';
```

See [Import Cacao CSS](/import) for more information.
