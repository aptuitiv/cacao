---
title: Grid columns - seven column
---

This provides styles for a seven-column grid.

Below is a basic example:

```html
<div class="grid">
    <div class="col-3-7">Column 1</div>
    <div class="col-4-7">Column 2</div>
</div>
```

The following sizes are available:

- col-1-7
- col-2-7
- col-3-7
- col-4-7
- col-5-7
- col-6-7
- col-7-7

Each column size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
col-1-7-3xs
col-1-7-2xs
col-1-7-xs
col-1-7-sm
col-1-7-md
col-1-7-lg
col-1-7-xl
col-1-7-2xl
col-1-7-3xl
col-1-7-4xl
col-1-7-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-7.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-7.css';
@import 'cacao-css/dist/grid-column/2xs/col-7.css';
@import 'cacao-css/dist/grid-column/xs/col-7.css';
@import 'cacao-css/dist/grid-column/sm/col-7.css';
@import 'cacao-css/dist/grid-column/md/col-7.css';
@import 'cacao-css/dist/grid-column/lg/col-7.css';
@import 'cacao-css/dist/grid-column/xl/col-7.css';
@import 'cacao-css/dist/grid-column/2xl/col-7.css';
@import 'cacao-css/dist/grid-column/3xl/col-7.css';
@import 'cacao-css/dist/grid-column/4xl/col-7.css';
@import 'cacao-css/dist/grid-column/5xl/col-7.css';
```

See [Import Cacao CSS](/import) for more information.
