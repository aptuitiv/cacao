---
title: Grid columns - ten column
---

This provides styles for a ten-column grid.

Below is a basic example:

```html
<div class="grid">
    <div class="col-3-10">Column 1</div>
    <div class="col-7-10">Column 2</div>
</div>
```

The following sizes are available:

- col-1-10
- col-2-10
- col-3-10
- col-4-10
- col-5-10
- col-6-10
- col-7-10
- col-8-10
- col-9-10
- col-10-10

Each column size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
col-1-10-3xs
col-1-10-2xs
col-1-10-xs
col-1-10-sm
col-1-10-md
col-1-10-lg
col-1-10-xl
col-1-10-2xl
col-1-10-3xl
col-1-10-4xl
col-1-10-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-10.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-10.css';
@import 'cacao-css/dist/grid-column/2xs/col-10.css';
@import 'cacao-css/dist/grid-column/xs/col-10.css';
@import 'cacao-css/dist/grid-column/sm/col-10.css';
@import 'cacao-css/dist/grid-column/md/col-10.css';
@import 'cacao-css/dist/grid-column/lg/col-10.css';
@import 'cacao-css/dist/grid-column/xl/col-10.css';
@import 'cacao-css/dist/grid-column/2xl/col-10.css';
@import 'cacao-css/dist/grid-column/3xl/col-10.css';
@import 'cacao-css/dist/grid-column/4xl/col-10.css';
@import 'cacao-css/dist/grid-column/5xl/col-10.css';
```

See [Import Cacao CSS](/import) for more information.
