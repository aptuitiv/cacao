---
title: Height - halfs
---

This provides styles for heights using half sizes.

The following sizes are available:

- h-1-2
- h-2-2

Each height size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
h-1-2-3xs
h-1-2-2xs
h-1-2-xs
h-1-2-sm
h-1-2-md
h-1-2-lg
h-1-2-xl
h-1-2-2xl
h-1-2-3xl
h-1-2-4xl
h-1-2-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/height/height-2.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/height/3xs/height-2.css';
@import 'cacao-css/dist/height/2xs/height-2.css';
@import 'cacao-css/dist/height/xs/height-2.css';
@import 'cacao-css/dist/height/sm/height-2.css';
@import 'cacao-css/dist/height/md/height-2.css';
@import 'cacao-css/dist/height/lg/height-2.css';
@import 'cacao-css/dist/height/xl/height-2.css';
@import 'cacao-css/dist/height/2xl/height-2.css';
@import 'cacao-css/dist/height/3xl/height-2.css';
@import 'cacao-css/dist/height/4xl/height-2.css';
@import 'cacao-css/dist/height/5xl/height-2.css';
```
