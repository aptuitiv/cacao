---
title: Full height
---

This provides styles for setting an element to be full height.

The following sizes are available and are the same:

- h-100
- h-1-1

They both set the height to `100%`.

Each height size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
h-1-1-3xs
h-1-1-2xs
h-1-1-xs
h-1-1-sm
h-1-1-md
h-1-1-lg
h-1-1-xl
h-1-1-1xl
h-1-1-3xl
h-1-1-4xl
h-1-1-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/height/height-1.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/height/3xs/height-1.css';
@import 'cacao-css/dist/height/2xs/height-1.css';
@import 'cacao-css/dist/height/xs/height-1.css';
@import 'cacao-css/dist/height/sm/height-1.css';
@import 'cacao-css/dist/height/md/height-1.css';
@import 'cacao-css/dist/height/lg/height-1.css';
@import 'cacao-css/dist/height/xl/height-1.css';
@import 'cacao-css/dist/height/2xl/height-1.css';
@import 'cacao-css/dist/height/3xl/height-1.css';
@import 'cacao-css/dist/height/4xl/height-1.css';
@import 'cacao-css/dist/height/5xl/height-1.css';
```
