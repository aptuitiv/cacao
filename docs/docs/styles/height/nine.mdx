---
title: Height - nineths
---

This provides styles for heights using ninths.

The following sizes are available:

- h-1-9
- h-2-9
- h-3-9
- h-4-9
- h-5-9
- h-6-9
- h-7-9
- h-8-9
- h-9-9

Each height size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
h-1-9-3xs
h-1-9-2xs
h-1-9-xs
h-1-9-sm
h-1-9-md
h-1-9-lg
h-1-9-xl
h-1-9-2xl
h-1-9-3xl
h-1-9-4xl
h-1-9-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/height/height-9.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/height/3xs/height-9.css';
@import 'cacao-css/dist/height/2xs/height-9.css';
@import 'cacao-css/dist/height/xs/height-9.css';
@import 'cacao-css/dist/height/sm/height-9.css';
@import 'cacao-css/dist/height/md/height-9.css';
@import 'cacao-css/dist/height/lg/height-9.css';
@import 'cacao-css/dist/height/xl/height-9.css';
@import 'cacao-css/dist/height/2xl/height-9.css';
@import 'cacao-css/dist/height/3xl/height-9.css';
@import 'cacao-css/dist/height/4xl/height-9.css';
@import 'cacao-css/dist/height/5xl/height-9.css';
```
