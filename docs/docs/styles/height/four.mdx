---
title: Height - fourths
---

This provides styles for heights using fourths.

The following sizes are available:

- h-1-4
- h-2-4
- h-3-4
- h-4-4

Each height size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
h-1-4-3xs
h-1-4-2xs
h-1-4-xs
h-1-4-sm
h-1-4-md
h-1-4-lg
h-1-4-xl
h-1-4-2xl
h-1-4-3xl
h-1-4-4xl
h-1-4-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/height/height-4.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/height/3xs/height-4.css';
@import 'cacao-css/dist/height/2xs/height-4.css';
@import 'cacao-css/dist/height/xs/height-4.css';
@import 'cacao-css/dist/height/sm/height-4.css';
@import 'cacao-css/dist/height/md/height-4.css';
@import 'cacao-css/dist/height/lg/height-4.css';
@import 'cacao-css/dist/height/xl/height-4.css';
@import 'cacao-css/dist/height/2xl/height-4.css';
@import 'cacao-css/dist/height/3xl/height-4.css';
@import 'cacao-css/dist/height/4xl/height-4.css';
@import 'cacao-css/dist/height/5xl/height-4.css';
```
