title: Height

This provides classes for setting the height of elements.

The following heights are available:

- [Auto height](/styles/height/auto)
- [Full height](/styles/height/one)
- [Halfs](/styles/height/two)
- [Thirds](/styles/height/three)
- [Fourths](/styles/height/four)
- [Fifths](/styles/height/five)
- [Sixths](/styles/height/six)
- [Sevenths](/styles/height/seven)
- [Eighths](/styles/height/eight)
- [Nineths](/styles/height/nine)
- [Tenths](/styles/height/ten)

## Combined imports

If you will use all of the height sizes then you can use the combined import file.

```css
@import 'cacao-css/dist/height/combined-import.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/height/3xs/combined-import.css';
@import 'cacao-css/dist/height/2xs/combined-import.css';
@import 'cacao-css/dist/height/xs/combined-import.css';
@import 'cacao-css/dist/height/sm/combined-import.css';
@import 'cacao-css/dist/height/md/combined-import.css';
@import 'cacao-css/dist/height/lg/combined-import.css';
@import 'cacao-css/dist/height/xl/combined-import.css';
@import 'cacao-css/dist/height/2xl/combined-import.css';
@import 'cacao-css/dist/height/3xl/combined-import.css';
@import 'cacao-css/dist/height/4xl/combined-import.css';
@import 'cacao-css/dist/height/5xl/combined-import.css';
```
