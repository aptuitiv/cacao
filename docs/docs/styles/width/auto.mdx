---
title: Auto width
---

This provides styles for setting the width of an element to be `auto`.

The following class is available:

- w-auto

The width has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
w-auto-3xs
w-auto-2xs
w-auto-xs
w-auto-sm
w-auto-md
w-auto-lg
w-auto-xl
w-auto-1xl
w-auto-3xl
w-auto-4xl
w-auto-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/width/width-auto.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/width/3xs/width-auto.css';
@import 'cacao-css/dist/width/2xs/width-auto.css';
@import 'cacao-css/dist/width/xs/width-auto.css';
@import 'cacao-css/dist/width/sm/width-auto.css';
@import 'cacao-css/dist/width/md/width-auto.css';
@import 'cacao-css/dist/width/lg/width-auto.css';
@import 'cacao-css/dist/width/xl/width-auto.css';
@import 'cacao-css/dist/width/2xl/width-auto.css';
@import 'cacao-css/dist/width/3xl/width-auto.css';
@import 'cacao-css/dist/width/4xl/width-auto.css';
@import 'cacao-css/dist/width/5xl/width-auto.css';
```

See [Import Cacao CSS](/import) for more information.
