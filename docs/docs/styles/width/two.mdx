---
title: Width - halfs
---

This provides styles for widths using half sizes.

The following sizes are available:

- w-1-2
- w-2-2

Each width size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
w-1-2-3xs
w-1-2-2xs
w-1-2-xs
w-1-2-sm
w-1-2-md
w-1-2-lg
w-1-2-xl
w-1-2-2xl
w-1-2-3xl
w-1-2-4xl
w-1-2-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/width/width-2.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/width/3xs/width-2.css';
@import 'cacao-css/dist/width/2xs/width-2.css';
@import 'cacao-css/dist/width/xs/width-2.css';
@import 'cacao-css/dist/width/sm/width-2.css';
@import 'cacao-css/dist/width/md/width-2.css';
@import 'cacao-css/dist/width/lg/width-2.css';
@import 'cacao-css/dist/width/xl/width-2.css';
@import 'cacao-css/dist/width/2xl/width-2.css';
@import 'cacao-css/dist/width/3xl/width-2.css';
@import 'cacao-css/dist/width/4xl/width-2.css';
@import 'cacao-css/dist/width/5xl/width-2.css';
```

See [Import Cacao CSS](/import) for more information.
