---
title: Full width
---

This provides styles for setting an element to be full width.

The following sizes are available and are the same:

- w-100
- w-1-1

They both set the width to `100%`.

Each width size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
w-1-1-3xs
w-1-1-2xs
w-1-1-xs
w-1-1-sm
w-1-1-md
w-1-1-lg
w-1-1-xl
w-1-1-1xl
w-1-1-3xl
w-1-1-4xl
w-1-1-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/width/width-1.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/width/3xs/width-1.css';
@import 'cacao-css/dist/width/2xs/width-1.css';
@import 'cacao-css/dist/width/xs/width-1.css';
@import 'cacao-css/dist/width/sm/width-1.css';
@import 'cacao-css/dist/width/md/width-1.css';
@import 'cacao-css/dist/width/lg/width-1.css';
@import 'cacao-css/dist/width/xl/width-1.css';
@import 'cacao-css/dist/width/2xl/width-1.css';
@import 'cacao-css/dist/width/3xl/width-1.css';
@import 'cacao-css/dist/width/4xl/width-1.css';
@import 'cacao-css/dist/width/5xl/width-1.css';
```

See [Import Cacao CSS](/import) for more information.
