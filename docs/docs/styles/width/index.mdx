This provides classes for setting the width of elements.

The following widths are available:

- [Auto width](/styles/width/auto)
- [Full width](/styles/width/one)
- [Halfs](/styles/width/two)
- [Thirds](/styles/width/three)
- [Fourths](/styles/width/four)
- [Fifths](/styles/width/five)
- [Sixths](/styles/width/six)
- [Sevenths](/styles/width/seven)
- [Eighths](/styles/width/eight)
- [Nineths](/styles/width/nine)
- [Tenths](/styles/width/ten)

## Combined imports

If you will use all of the width sizes then you can use the combined import file.

```css
@import 'cacao-css/dist/width/combined-import.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/width/3xs/combined-import.css';
@import 'cacao-css/dist/width/2xs/combined-import.css';
@import 'cacao-css/dist/width/xs/combined-import.css';
@import 'cacao-css/dist/width/sm/combined-import.css';
@import 'cacao-css/dist/width/md/combined-import.css';
@import 'cacao-css/dist/width/lg/combined-import.css';
@import 'cacao-css/dist/width/xl/combined-import.css';
@import 'cacao-css/dist/width/2xl/combined-import.css';
@import 'cacao-css/dist/width/3xl/combined-import.css';
@import 'cacao-css/dist/width/4xl/combined-import.css';
@import 'cacao-css/dist/width/5xl/combined-import.css';
```
