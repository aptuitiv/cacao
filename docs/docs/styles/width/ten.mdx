---
title: Width - tenths
---

This provides styles for widths using tenths.

The following sizes are available:

- w-1-10
- w-2-10
- w-3-10
- w-4-10
- w-5-10
- w-6-10
- w-7-10
- w-8-10
- w-9-10
- w-10-10

Each width size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
w-1-10-3xs
w-1-10-2xs
w-1-10-xs
w-1-10-sm
w-1-10-md
w-1-10-lg
w-1-10-xl
w-1-10-2xl
w-1-10-3xl
w-1-10-4xl
w-1-10-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/width/width-10.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/width/3xs/width-10.css';
@import 'cacao-css/dist/width/2xs/width-10.css';
@import 'cacao-css/dist/width/xs/width-10.css';
@import 'cacao-css/dist/width/sm/width-10.css';
@import 'cacao-css/dist/width/md/width-10.css';
@import 'cacao-css/dist/width/lg/width-10.css';
@import 'cacao-css/dist/width/xl/width-10.css';
@import 'cacao-css/dist/width/2xl/width-10.css';
@import 'cacao-css/dist/width/3xl/width-10.css';
@import 'cacao-css/dist/width/4xl/width-10.css';
@import 'cacao-css/dist/width/5xl/width-10.css';
```
