---
title: Grid columns - three column
---

This provides styles for a three-column grid.

The following sizes are available:

- col-1-3
- col-2-3
- col-3-3

Each column size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
col-1-3-3xs
col-1-3-2xs
col-1-3-xs
col-1-3-sm
col-1-3-md
col-1-3-lg
col-1-3-xl
col-1-3-2xl
col-1-3-3xl
col-1-3-4xl
col-1-3-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-3.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-3.css';
@import 'cacao-css/dist/grid-column/2xs/col-3.css';
@import 'cacao-css/dist/grid-column/xs/col-3.css';
@import 'cacao-css/dist/grid-column/sm/col-3.css';
@import 'cacao-css/dist/grid-column/md/col-3.css';
@import 'cacao-css/dist/grid-column/lg/col-3.css';
@import 'cacao-css/dist/grid-column/xl/col-3.css';
@import 'cacao-css/dist/grid-column/2xl/col-3.css';
@import 'cacao-css/dist/grid-column/3xl/col-3.css';
@import 'cacao-css/dist/grid-column/4xl/col-3.css';
@import 'cacao-css/dist/grid-column/5xl/col-3.css';
```
