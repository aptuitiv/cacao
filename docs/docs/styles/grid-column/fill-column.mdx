---
title: Grid columns - fill column width
---

This provides styles for a column to fill the remaining space in the grid. You can use this on one or more columns in a grid.


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-fill.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-fill.css';
@import 'cacao-css/dist/grid-column/2xs/col-fill.css';
@import 'cacao-css/dist/grid-column/xs/col-fill.css';
@import 'cacao-css/dist/grid-column/sm/col-fill.css';
@import 'cacao-css/dist/grid-column/md/col-fill.css';
@import 'cacao-css/dist/grid-column/lg/col-fill.css';
@import 'cacao-css/dist/grid-column/xl/col-fill.css';
@import 'cacao-css/dist/grid-column/2xl/col-fill.css';
@import 'cacao-css/dist/grid-column/3xl/col-fill.css';
@import 'cacao-css/dist/grid-column/4xl/col-fill.css';
@import 'cacao-css/dist/grid-column/5xl/col-fill.css';
```
