---
title: Grid columns - four column
---

This provides styles for a four-column grid.

The following sizes are available:

- col-1-4
- col-2-4
- col-3-4
- col-4-4

Each column size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
col-1-4-3xs
col-1-4-2xs
col-1-4-xs
col-1-4-sm
col-1-4-md
col-1-4-lg
col-1-4-xl
col-1-4-2xl
col-1-4-3xl
col-1-4-4xl
col-1-4-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-4.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-4.css';
@import 'cacao-css/dist/grid-column/2xs/col-4.css';
@import 'cacao-css/dist/grid-column/xs/col-4.css';
@import 'cacao-css/dist/grid-column/sm/col-4.css';
@import 'cacao-css/dist/grid-column/md/col-4.css';
@import 'cacao-css/dist/grid-column/lg/col-4.css';
@import 'cacao-css/dist/grid-column/xl/col-4.css';
@import 'cacao-css/dist/grid-column/2xl/col-4.css';
@import 'cacao-css/dist/grid-column/3xl/col-4.css';
@import 'cacao-css/dist/grid-column/4xl/col-4.css';
@import 'cacao-css/dist/grid-column/5xl/col-4.css';
```

See [Import Cacao CSS](/import) for more information.
