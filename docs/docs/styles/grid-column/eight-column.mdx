---
title: Grid columns - 8 column
---

This provides styles for a eight-column grid.

The following sizes are available:

- col-1-8
- col-2-8
- col-3-8
- col-4-8
- col-5-8
- col-6-8
- col-7-8
- col-8-8

Each column size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
col-1-8-3xs
col-1-8-2xs
col-1-8-xs
col-1-8-sm
col-1-8-md
col-1-8-lg
col-1-8-xl
col-1-8-2xl
col-1-8-3xl
col-1-8-4xl
col-1-8-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-8.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-8.css';
@import 'cacao-css/dist/grid-column/2xs/col-8.css';
@import 'cacao-css/dist/grid-column/xs/col-8.css';
@import 'cacao-css/dist/grid-column/sm/col-8.css';
@import 'cacao-css/dist/grid-column/md/col-8.css';
@import 'cacao-css/dist/grid-column/lg/col-8.css';
@import 'cacao-css/dist/grid-column/xl/col-8.css';
@import 'cacao-css/dist/grid-column/2xl/col-8.css';
@import 'cacao-css/dist/grid-column/3xl/col-8.css';
@import 'cacao-css/dist/grid-column/4xl/col-8.css';
@import 'cacao-css/dist/grid-column/5xl/col-8.css';
```
