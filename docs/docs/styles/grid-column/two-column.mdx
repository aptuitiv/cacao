---
title: Grid columns - two column
---

This provides styles for a two-column grid.

The following sizes are available:

- col-1-2
- col-2-2

Each column size has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
col-1-2-3xs
col-1-2-2xs
col-1-2-xs
col-1-2-sm
col-1-2-md
col-1-2-lg
col-1-2-xl
col-1-2-2xl
col-1-2-3xl
col-1-2-4xl
col-1-2-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/grid-column/col-2.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/grid-column/3xs/col-2.css';
@import 'cacao-css/dist/grid-column/2xs/col-2.css';
@import 'cacao-css/dist/grid-column/xs/col-2.css';
@import 'cacao-css/dist/grid-column/sm/col-2.css';
@import 'cacao-css/dist/grid-column/md/col-2.css';
@import 'cacao-css/dist/grid-column/lg/col-2.css';
@import 'cacao-css/dist/grid-column/xl/col-2.css';
@import 'cacao-css/dist/grid-column/2xl/col-2.css';
@import 'cacao-css/dist/grid-column/3xl/col-2.css';
@import 'cacao-css/dist/grid-column/4xl/col-2.css';
@import 'cacao-css/dist/grid-column/5xl/col-2.css';
```

See [Import Cacao CSS](/import) for more information.
