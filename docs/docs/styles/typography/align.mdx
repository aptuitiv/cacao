---
title: Text alignment
---

The align classes set the `text-align` styles.

Each text align base class has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
text-center-3xs
text-center-2xs
text-center-xs
text-center-sm
text-center-md
text-center-lg
text-center-xl
text-center-1xl
text-center-3xl
text-center-4xl
text-center-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/typography/align.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/typography/3xs/align.css';
@import 'cacao-css/dist/typography/2xs/align.css';
@import 'cacao-css/dist/typography/xs/align.css';
@import 'cacao-css/dist/typography/sm/align.css';
@import 'cacao-css/dist/typography/md/align.css';
@import 'cacao-css/dist/typography/lg/align.css';
@import 'cacao-css/dist/typography/xl/align.css';
@import 'cacao-css/dist/typography/2xl/align.css';
@import 'cacao-css/dist/typography/3xl/align.css';
@import 'cacao-css/dist/typography/4xl/align.css';
@import 'cacao-css/dist/typography/5xl/align.css';
```

See [Import Cacao CSS](/import) for more information.
