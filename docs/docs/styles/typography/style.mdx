---
title: Font styles
---

The font style classes set the `font-style` style.

Each font style base class has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
style-italic-3xs
style-italic-2xs
style-italic-xs
style-italic-sm
style-italic-md
style-italic-lg
style-italic-xl
style-italic-1xl
style-italic-3xl
style-italic-4xl
style-italic-5xl
```

## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/typography/style.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/typography/3xs/style.css';
@import 'cacao-css/dist/typography/2xs/style.css';
@import 'cacao-css/dist/typography/xs/style.css';
@import 'cacao-css/dist/typography/sm/style.css';
@import 'cacao-css/dist/typography/md/style.css';
@import 'cacao-css/dist/typography/lg/style.css';
@import 'cacao-css/dist/typography/xl/style.css';
@import 'cacao-css/dist/typography/2xl/style.css';
@import 'cacao-css/dist/typography/3xl/style.css';
@import 'cacao-css/dist/typography/4xl/style.css';
@import 'cacao-css/dist/typography/5xl/style.css';
```

See [Import Cacao CSS](/import) for more information.
