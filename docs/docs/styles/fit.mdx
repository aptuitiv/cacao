---
title: Object ft
---

This provides styles for setting the `object-fit` style for elements.

Each object fit base class has a class to go with each of the [media query viewport sizes](/styles/media#viewport-media-queries). For example:

```css
fit-cover-3xs
fit-cover-2xs
fit-cover-xs
fit-cover-sm
fit-cover-md
fit-cover-lg
fit-cover-xl
fit-cover-1xl
fit-cover-3xl
fit-cover-4xl
fit-cover-5xl
```


## How to import into your CSS

Import this into your CSS with:

```css
@import 'cacao-css/dist/fit/fit.css';
```

Imports are available for all [media query viewport sizes](/styles/media#viewport-media-queries).

```css
@import 'cacao-css/dist/fit/3xs/fit.css';
@import 'cacao-css/dist/fit/2xs/fit.css';
@import 'cacao-css/dist/fit/xs/fit.css';
@import 'cacao-css/dist/fit/sm/fit.css';
@import 'cacao-css/dist/fit/md/fit.css';
@import 'cacao-css/dist/fit/lg/fit.css';
@import 'cacao-css/dist/fit/xl/fit.css';
@import 'cacao-css/dist/fit/2xl/fit.css';
@import 'cacao-css/dist/fit/3xl/fit.css';
@import 'cacao-css/dist/fit/4xl/fit.css';
@import 'cacao-css/dist/fit/5xl/fit.css';
```

See [Import Cacao CSS](/import) for more information.
